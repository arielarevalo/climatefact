# 3.3 Mitigation Pathways

Limiting human-caused global warming requires net zero anthropogenic $\mathbf { C O } _ { 2 }$ emissions. Pathways consistent with $1 . 5 \textdegree$ and ${ \sf z } ^ { \circ } { \sf C }$ carbon budgets imply rapid, deep, and in most cases immediate GHG emission reductions in all sectors (high confidence). Exceeding a warming level and returning (i.e. overshoot) implies increased risks and potential irreversible impacts; achieving and sustaining global net negative $\mathbf { C O } _ { 2 }$ emissions would reduce warming (high confidence).
